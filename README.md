### 🏆 Projets Phares

#### 1. 🏎️ Vision Solver Bot (C++ / Python / OpenCV)
> *L'optimisation à l'état pur.*
Un bot capable de résoudre des tâches visuelles complexes en **moins de 100ms**.
- **Tech :** Moteur C++ avec Bitboards 64-bits, Algorithme Beam Search, Multi-threading OpenMP.
- **Vision :** Traitement d'image temps réel avec OpenCV.
- **Performance :** Binding Python via `ctypes` pour une intégration fluide.

#### 2. 🎨 Configurateur AVG (React / Firebase)
> *Architecture SPA complexe.*
Une application web permettant la configuration de produits sur-mesure avec synchronisation temps réel.
- **Tech :** React, Tailwind CSS, Firebase Firestore.
- **Features :** Gestion d'états globaux, sérialisation JSON pour partage de config (Presse-papier), Routing dynamique.

#### 3. 🏢 Migration Odoo (Python / XML)
> *Expérience professionnelle.*
Pilotage de la migration de parcs clients vers Odoo 18.
- **Tech :** Python, PostgreSQL, Framework OWL 2.
- **Réalisation :** Modules Helpdesk custom et rapports QWeb dynamiques.
